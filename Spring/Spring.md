### 1. Bean 的作用域

| 类别        | 说明                                        |
| --------- | ----------------------------------------- |
| singleton | 默认在 Spring 容器中仅存在一个实例                     |
| prototype | 每次调用 getBean() 都重新生成一个实例                  |
| request   | 为每个 HTTP 请求生成一个实例                         |
| session   | 同一个 HTTP session 使用一个实例，不同 session 使用不同实例 |
